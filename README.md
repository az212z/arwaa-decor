# أروع لمسات للديكور — موقع تعريفي

موقع تسويقي (Landing page) عربي RTL لمعرض **ديكورات أروع لمسات للديكور**.

## النشاط
ديكورات وتشطيبات: جبس، ورق جدران، تكسيات خشبية، بديل رخام، باركيه، ديكورات جبسية، براويز، ستائر، وأبواب سحب.

## المدينة
بريدة — حي الريان.

## سبب الاختيار
نشاط بصري بامتياز (أعمال تنفيذ حقيقية تُظهر جودة الحرفة)، تقييم قوي **4.9 من ٦٠٩ مراجعة** على خرائط قوقل، وبدون موقع إلكتروني — فرصة واضحة لتحويل الزوار إلى طلبات عبر واتساب.

## فكرة التصميم
اتجاه «الحرفة والتفاصيل»: فحمي دافئ (#241F1B) + ذهبي نحاسي (#B89150) + كريمي (#F3EEE6). معرض أعمال هو محور الصفحة، خط عرض **El Messiri** مع متن **Tajawal**، حركة يدوية خفيفة (كشف عند التمرير، تكبير الصور، رأس يتقلّص).

## التقنيات
HTML5 + CSS3 + JavaScript (vanilla) — بدون أي مكتبات أو خطوة بناء. خطوط قوقل عبر `<link>`، صور بتحميل كسول `loading="lazy"`.

## الملفات
- `index.html` — الصفحة الرئيسية
- `404.html` — صفحة الخطأ
- `assets/css/style.css` — التنسيقات
- `assets/js/main.js` — التفاعل (القائمة، الكشف، النموذج، الـ lightbox)
- `assets/img/` — صور حقيقية للنشاط
- `favicon.svg` — الأيقونة

## الصور المستخدمة
- `aw-27876.jpg` — جدار بديل رخام + تكسية خشبية (هيرو + معرض)
- `aw-25426.jpg` — براويز وإطارات ذهبية
- `aw-1373.jpg` — ورق جدران فني وأقواس جبسية
- `aw-11647.jpg` — تنفيذ بديل رخام (قسم لماذا نحن + معرض)
- `aw-7230.jpg` — واجهة المعرض في بريدة (الموقع + معرض)

> استُبعدت صورتان تحملان علامة مائية لجهة/رقم مختلف (`aw-5841.jpg`, `aw-9881.jpg`) حفاظًا على هوية النشاط.

**مصدر الصور:** خرائط قوقل (ملف النشاط الحقيقي).

## حالة الـ Backend
غير مربوط — نموذج طلب عرض السعر **تجريبي**: يجهّز رسالة واتساب جاهزة (`wa.me/966530676558`)، ويحفظ نسخة محليًا في `localStorage`، ويعرض إشعار نجاح. لا توجد قاعدة بيانات أو خادم.

## التشغيل محليًا
```bash
python3 -m http.server 8000
# ثم افتح: http://localhost:8000
```
